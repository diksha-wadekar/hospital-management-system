# 🏥 Serenity Care Hospital — Management System

A frontend hospital management web application built with pure HTML, CSS, and JavaScript. Designed to simulate real hospital workflows including patient registration, status tracking, and information lookup — all within the browser using local storage.

## 🖥️ Live Preview

> Run locally by opening `index.html` in any browser. No installation required.

## ✨ Features

- **Patient Registration** — Add new patients with details like name, condition, doctor, and room
- **Patient Status** — View and update the current status of admitted patients
- **Patient Info** — Search and retrieve detailed patient records by name, condition, or doctor
- **Hospital Dashboard** — Overview of departments and services
- **Search Bar** — Real-time search across patient records
- **Persistent Local Storage** — Patient data is saved in the browser across sessions

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Structure | HTML5 |
| Styling | CSS3 |
| Logic | Vanilla JavaScript |
| Storage | Browser LocalStorage |

## 🚀 Getting Started

```bash

git clone https://github.com/diksha-wadekar/serenity-care-hospital.git

```
No dependencies. No build step. Just open and run.
